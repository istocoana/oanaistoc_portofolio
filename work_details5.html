Futureship UX/UI

This UI/UX design is crafted for Futureship, a personal project designed for the Innovation Labs incubator. It's tailored to streamline the internship search and application process, with a focus on a clean, user-friendly interface that enhances user engagement and simplifies navigation. With an intuitive layout, the platform is created to encourage students to explore and apply to various internships, fostering a productive and engaging experience for its user base.

  • Clean and Intuitive Layout: The design uses a straightforward layout with a navigation bar on the left, providing clear access to various user functions like "My profile," "Saved Internships," "View CV," and "Settings." This promotes an intuitive user journey within the app.
  • Personalized User Dashboard: The main profile page is personalized with the user's name and a cheerful profile picture, alongside essential contact information, including phone, email, and social links to platforms like GitHub. This personalization enhances user engagement.
  • Interactive Elements: Elements like drop-downs for education details, plus signs for adding skills, and checkboxes for filters in the internship search enhance interactivity, providing a dynamic UX.
  • Information Organization: Data is neatly organized into tabs and sections for easy consumption. For example, the University and Volunteering sections on the profile page help in segmenting personal achievements and experiences, making the information digestible and easy to navigate.
